Repositório criado para acompanhar minhas aulas e práticas do curso de Git e GitHub do Curso em Vídeo.

---

## 📖 Sobre

Aqui estão armazenados os exercícios, anotações e testes que fiz durante o curso.  
O objetivo é praticar versionamento de código e aprender a usar o Git e o GitHub na prática.

---

## 🛠️ Tecnologias utilizadas

- Git
- GitHub
- Markdown

---

## 📌 O que aprendi

- Criar e clonar repositórios
- Fazer commits e entender o histórico
- Usar `git push` e `git pull`
- Trabalhar com branches
- Resolver conflitos simples
- Criar README.md
